# Code Review Check

- [ ] Code follows controller/service/module separation shown in `src/`.
- [ ] New domain code is registered through Nest module metadata.
- [ ] Public methods/contracts are typed; no new `any`.
- [ ] Promises are awaited, handled, or explicitly voided.
- [ ] Prettier conventions and repository naming are preserved.
- [ ] `pnpm run build` and relevant tests pass.
